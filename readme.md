# To Do List API
## Repositório que contêm o CRUD completo da aplicação

### Configuração
Na pasta ``/config``, seguir o exemplo do arquivo config.Example.json e alterar o arquivo ``/config.json`` com as configurações de seu banco de dados
### Como rodar
```
npm install
npm start
```

### Tecnologias
- Nodejs
- Express
- Sequelize
